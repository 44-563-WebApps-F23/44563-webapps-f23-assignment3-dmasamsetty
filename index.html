Coorg

Coorg has some very good high mountain view points and is also very pleasant place to spend the Vacation. We can also have some great trekking experience. Nowadays it is one of the famous tourist place to visit. The beauty of the nature there wont let you go back. You can also see some popular plantations like tea and coffee.

Activities to do in Coorg

  1. Sight Seeing
  2. Trekking
  3. Camping

Some of the great dishes in Coorg that you can get are:

  • Bamboo Shoot Curry
  • Coorg Chicken Curry
  • Kuleputtu

@function to-fixed($float, $digits: 2) {
                $sass-precision: 5;
                
                @if $digits > $sass-precision {
                  @warn "Sass sets default precision to #{$sass-precision} digits, and there is no way to change that for now."
                  + "The returned number will have #{$sass-precision} digits, even if you asked for `#{$digits}`."
                  + "See https://github.com/sass/sass/issues/1122 for further informations.";
                }
                
                $pow: pow(10, $digits);
                @return round($float * $pow) / $pow;
              }
